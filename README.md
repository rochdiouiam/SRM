# SRM
#  Société Régionale Multiservices de l’Oriental (SRM L’Oriental SA)
Ce projet est un **site web statique** développé dans un cadre académique.  
Il représente une plateforme web de la **Société Régionale Multiservices de l’Oriental (SRM L’Oriental SA)**, visant à présenter l’entreprise et offrir des pages d’interaction basiques aux utilisateurs.

# SRM mini projet git et github

#  Société Régionale Multiservices de l’Oriental (SRM L’Oriental SA)
Ce projet est un **site web statique** développé dans un cadre académique.  
Il représente une plateforme web de la **Société Régionale Multiservices de l’Oriental (SRM L’Oriental SA)**, visant à présenter l’entreprise et offrir des pages d’interaction basiques aux utilisateurs.

## À propos de SRM L’Oriental

La **Société Régionale Multiservices de l’Oriental (SRM)** est une société chargée de la gestion de plusieurs services publics au niveau de la région de l’Oriental.

##  Technologies utilisées

- **HTML5** : structure des pages web  
- **CSS3** : design et mise en page  
- **JavaScript** : interactions côté client

##  Fonctionnalités

- Navigation entre plusieurs pages
- Interface utilisateur simple et intuitive
- Intégration du logo et des images de l’entreprise
- Séparation claire entre structure, style et scripts
